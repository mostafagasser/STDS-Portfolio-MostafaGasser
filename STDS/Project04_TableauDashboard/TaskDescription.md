## Task Objective
Build an interactive HR dashboard using Tableau to analyze employee attrition
and identify key factors affecting employee turnover.

## Dataset Description
The dataset contains employee-level HR data including demographics,
job roles, satisfaction metrics, and attrition status.

### Key Columns Used
- Attrition
- Department
- Gender
- Age
- Education Field
- Job Role
- Job Satisfaction
- Employee Count

## Dashboard Features
- KPI cards for overall workforce metrics
- Attrition analysis by department, gender, age group, and education field
- Interactive filters for Education and Department
- Visual comparison of job satisfaction across job roles

## KPIs Included
- Total Employee Count
- Attrition Count
- Active Employees
- Attrition Rate
- Average Employee Age

## Tools Used
- Tableau
