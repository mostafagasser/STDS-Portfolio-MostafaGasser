## Task Objective
Design an interactive Excel dashboard to analyze project performance,
profitability, and operational metrics to support management decision-making.

## Dashboard Overview
The dashboard provides a comprehensive view of project performance including:
- Project profitability
- Accepted vs rejected projects
- Cost and sales analysis
- Project distribution by government entity and region
- Monthly and yearly project trends

## Key KPIs
- Total profit
- Number of accepted projects
- Number of rejected projects
- Total sales
- Total costs
- Profit margin

## Dashboard Features
- Interactive slicers for project name, government entity, and date
- Dynamic charts and KPIs
- Professional layout and color theme

## Tools Used
- Microsoft Excel
